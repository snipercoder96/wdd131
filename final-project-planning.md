# A blog on HTML, CSS, JavaScript

## Home Page
- `Header`: Blog title + navigation bar.

- `Hero Section`: Short intro text (“Beginner-friendly coding tips”).

- Featured Tip Box: Highlight one coding tip (later powered by JS).

- `Footer`: Current year + last modified date (JS-driven).

## Tips Page
- `Grid`/List Layout: Cards for each tip.

- Card structure: Title → short description → “Read more” link.

- Space for code snippets (monospace font).

- Filter Section: Buttons to filter tips by category (HTML, CSS, JS).



## Resources Page
- List of Links: External tutorials, docs, videos.

- Suggestion Form: Visitors can suggest topics (later validated with JS).


## About Page
- Bio Section: Who you are + why you built the blog.

- Image/Icon: Adds personality.

- JS Current year and last modified.
