# Functions

## Syntax

``` javascript
function functionName(parameter1 , parameter2) { // Parameters
    // Returns a value (if needed)
  return `${parameter1} ${parameter2}`; // String Interpolation
}

```

Functions are great for modularizing code when code becomes more complex.

# Shorthand Javascript (arrow function)
``` javascript

const fullName = (firstName, lastName) => `${firstName} ${lastName}`;

```
=> means function() in shorthand
- (firstName, lastName) are the parameters of the => (arrow function)
