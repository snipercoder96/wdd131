# Set Item
``` javascript
localStorage.setItem(key, value);
```
- This sets the key, value pairs 

# Get Item

``` javascript
localStorage.getItem(key);

```
- This gets the VALUE of the key (think of a dictionary const name = {"name" : "Kevin"} => name["name"] //Result Kevin)

# Remove Item

``` javascript
localStorage.removeItem(key);
```

- Removes both {"key" : "value"} pairs (In other words it perishes)

# Clear
``` javascript
localStorage.clear();
```
- Removes all key value pairs depending on the number of Key value pairs -> {"name" : "Kevin", "age" : "18", "Gender" : "Male"}

# Length
``` javascript
localStorage.Length;
```
- Gets the length of an item (an item is a key, value pair)

# Key(index)

``` javascript
localStorage.Key(indexNum);
```
- Gets the name of the key in a position (indexNum *Remember it starts at 0.)
